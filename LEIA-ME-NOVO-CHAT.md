# Como abrir o próximo chat · KA

> Siga esse passo-a-passo toda vez que abrir um novo chat do Claude pra continuar o projeto.

---

## Arquivos a anexar no novo chat

Anexe os **6 arquivos** abaixo em **toda nova conversa**:

1. **`kellyalbert-site-v64.zip`** — site completo na versão atual (34 MB)
2. **`INICIO-DE-CHAT-KA.md`** — fonte da verdade operacional
3. **`DECISOES.md`** — log de decisões
4. **`PENDENCIAS.md`** — o que está aberto
5. **`REGRAS-PROJETO.md`** — regras técnicas
6. **`KA-Identidade-Visual-Completa.md`** — guia visual completo (usado também pelo chat de e-mails)

---
